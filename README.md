# 📊 E-commerce Sales Analysis Dashboard (Excel)

## 📌 Project Overview

This project is an interactive Microsoft Excel dashboard built to analyze e-commerce sales data and generate business insights. It helps track sales performance, customer purchasing behavior, product popularity, shipping efficiency, and order trends through interactive visualizations.

The dashboard allows users to filter and explore data dynamically using slicers.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track order quantity and revenue
- Identify popular products
- Analyze customer purchasing behavior
- Monitor delivery performance
- Compare sales across order channels
- Build an interactive business dashboard
